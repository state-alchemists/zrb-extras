# Zrb-Extras

Additional tasks and utilities for Zrb.

# Installing zrb-extras

```bash
pip install zrb-extras
```

# Using zrb-extras

You can use `zrb-extras` by importing the package in your `zrb_init.py`

```python
from zrb_extras import hello

print(hello())
```

# Prerequisites

- Python
- Pip
- Zrb >= 0.0.90
